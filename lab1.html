Preguntas y respuestas

  1. ¿Cuál es la diferencia entre Internet y la World Wide Web? Internet es el medio y World Wide Web es quien accese al internet

  2. ¿Cuáles son las partes de un URL?
    • Protocolo
    • Subdominio
    • Dominio
    • TLD
    • Ruta
    • Parametro
    • Etiqueta

  3. ¿Cuál es el propósito de los métodos HTTP: GET, HEAD, POST, PUT, PATCH, DELETE?
    • HTTP: Realizar petición de datos o informacón
    • GET: Peticion para recuperar dato en especifico
    • HEAD: Manda header y hacer diagnostico
    • POST: Enviar informacion
    • PUT: Modificar todo el recurso
    • PATCH: Modifica solo una parte previamente especificada del recurso
    • DELETE: Eliminar un recurso

  4. ¿Qué método HTTP se debe utilizar al enviar un formulario HTML, por ejemplo cuando ingresas tu usuario y contraseña en algún sitio? GET Y POST ¿Por qué? GET->envia los datos. POST->los envia de manera cifrada

  5. ¿Qué método HTTP se utiliza cuando a través de un navegador web se accede a una página a través de un URL? GET

  6. Un servidor web devuelve una respuesta HTTP con código 200. ¿Qué significa esto? Significa que la peticion ha sido recibida con exito ¿Ocurrió algún error? al contrario, ha habido un exito

  7. ¿Es responsabilidad del desarrollador corregir un sitio web si un usuario reporta que intentó acceder al sitio y se encontró con un error 404? No ¿Por qué? Ya que es un error por parte del usuario, ya sea en su interet, acceso, etc.

  8. ¿Es responsabilidad del desarrollador corregir un sitio web si un usuario reporta que intentó acceder al sitio y se encontró con un error 500? Es correcto ¿Por qué? Porque es un error de acceso

  9. ¿Qué significa que un atributo HTML5 esté depreciado o desaprobado (deprecated)? Menciona algunos. Mas que nada significa que se estan haciendo uso etiquetas de presentacion, por ejemplo
    • menu -> ul
    • isindex -> input
    • applet -> object
    • dir -> ul
    • font -> style

  10. ¿Cuáles son las diferencias principales entre HTML 4 y HTML5? Una de las principales diferencias esque en html4 habia etiquetas para todo, mientras que en html5 solo se enfoca en el contendio, dehjando la parte de presentacion a CSS y JavaScript
    • Nuevos elementos -> footer, article, aside, figure, etc
    • Nuevos atributos -> media, ping, target, replace, etc
    • HTML5 requiere doctype

  11. ¿Qué componentes de estructura y estilo tiene una tabla?
    • table -> crear tabla
    • tr -> filas
    • th -> encabezado de las filas
    • td -> información de los no encabezados

      • para mas orden:
      • thead -> header de la taba
      • tbody -> cuerpo de la tabla
      • tfoot -> pie de pagina de la tabla

  12. ¿Cuáles son los principales controles de una forma HTML5?
    • input type="number" = campo de entrada para input numerico
    • input type="range" = barra deslizadora
    • input type="color" = selector de color
    • input type="date" = selector de fecha
